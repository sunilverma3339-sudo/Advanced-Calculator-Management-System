## Project Description

SecureCalculatorPro is a menu-driven Python calculator application. It uses
object-oriented programming to perform basic arithmetic operations, validates
user input, stores successful calculations in a history file, logs errors with
timestamps, and displays an analytics report.

## Features

- Addition, subtraction, multiplication, and division
- Menu-driven terminal interface
- Input validation with graceful error handling
- Custom exception classes for calculator-specific errors
- Use of `try`, `except`, `else`, and `finally`
- Successful calculation history saved in `calculation_history.txt`
- Error details saved in `error_log.txt` using Python's `logging` module
- Error report with timestamps
- Analytics report showing:
  - Total successful calculations
  - Total errors
  - Most common error type

## Concepts Used

- Python classes and objects
- Encapsulation through dedicated classes and methods
- Static methods
- Custom exceptions
- Exception handling
- File handling
- Logging module
- Dictionaries and collections
- Menu-driven programming

## How to Run

1. Make sure Python is installed.
2. Open a terminal or command prompt in the project folder.
3. Run the program:

```bash
python calculator.py
```

4. Use the menu options:

```text
1. Perform calculations
2. View calculation history
3. View error report
4. Exit
```

## Project Files

- `calculator.py` - Main Python source code
- `calculation_history.txt` - Stores successful calculations
- `error_log.txt` - Stores logged errors with timestamps
